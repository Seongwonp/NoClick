## 2. 문제 정의

### 현실

국내 블로그·리뷰 플랫폼의 광고성 게시물 비율은 갈수록 증가하고 있습니다.
네이버 블로그, 인스타그램, 쿠팡 등에서 "솔직한 후기"라고 적힌 글의 상당수가 원고료·협찬을 받고 작성된 마케팅 글입니다.

### 핵심 문제

> **광고 글은 나쁜 점을 쓰지 않는다.**

소비자는 후기를 읽어도 단점을 알 수 없습니다. 알지 못한 채로 소비 결정을 내립니다.

### 페르소나

**김지연 (28세, 직장인)**
- 점심 맛집 찾을 때마다 블로그 후기 10개 이상 읽음
- "솔직한 후기라 믿었는데 가보니 별로였던 경험" 여러 번
- 협찬 글인지 구분할 기준이 없어 매번 불안함

**박현우 (35세, 주부)**
- 쿠팡에서 후기 보고 제품 샀다가 반품한 경험 다수
- 별점 5개 리뷰가 몰려있으면 의심하지만 방법을 모름

## 3. 솔루션 — No-Click

### 핵심 아이디어

> **"쓰지 않은 것"으로 단점을 역추론한다**

광고 글이 숨기는 정보는 패턴이 있습니다.
No-Click은 리뷰 텍스트를 AI 수사관처럼 분석해 **결핍된 정보**를 추론합니다.

### 제공 정보

| 분석 항목 | 설명 |
|-----------|------|
| **광고 확률** | 0~100% — 70 이상이면 광고 판정 |
| **신뢰도 점수** | 후기 신뢰 가능성 |
| **광고성 문구 탐지** | 과장 표현, 협찬 부인, 단점 회피 문구 하이라이트 |
| **숨겨진 단점 추론** | 글이 언급하지 않은 단점을 역추론 + 근거 제시 |
| **숨겨진 의도** | 글쓴이의 실제 목적 |
| **종합 판정** | AI 수사관의 최종 한 줄 결론 |
| **탈광고 요약** | 광고 걷어낸 실제 정보 한 줄 요약 |

### 사용 흐름

```
① 후기 텍스트 복사
     ↓
② No-Click에 붙여넣기 + 플랫폼 선택 (네이버 / 쿠팡 / 인스타 / 기타)
     ↓
③ AI 분석 (Gemini Flash, 약 3~5초)
     ↓
④ 광고 확률, 숨겨진 단점, 종합 판정 확인
     ↓
⑤ 클릭할지 말지 결정
```

---

### 실제 분석 예시

**입력 텍스트** (네이버 블로그 후기)
```
드디어 방문한 망원 핫플! 사장님이 너무 친절하시고 음식도 역대급이에요.
원고료를 받지 않은 솔직한 후기입니다. 분위기도 너무 좋고 인생 맛집 등극!!
주차가 좀 그렇긴 한데 그래도 다들 꼭 가보세요. 대중교통으로 오시길 강추해요!
```

**No-Click 분석 결과**

| 항목 | 결과 |
|------|------|
| 🔴 광고 확률 | **89%** — 광고 판정 |
| ⭐ 신뢰도 | 11점 |
| 💡 숨겨진 의도 | 업체로부터 식사 제공 또는 원고료를 받고 작성한 바이럴 마케팅 글 |
| ⚖️ 종합 판정 | 광고성 개입 가능성이 높습니다. 방문 전 가격·주차를 별도 확인하세요. |
| 📝 탈광고 요약 | 핵심 정보(가격·주차)가 누락된 과장형 광고 의심 후기입니다. |

**탐지된 광고성 문구**

| 문구 | 유형 |
|------|------|
| "원고료를 받지 않은 솔직한 후기입니다" | 협찬 부인 |
| "역대급", "인생 맛집" | 과장 표현 |
| "주차가 좀 그렇긴 한데 그래도 괜찮아요" | 단점 회피 |

**추론된 숨겨진 단점**

| 단점 | 신뢰도 | 근거 |
|------|--------|------|
| 주차 사실상 불가 — 방문 시 이동 불편 감수 필요 | 92% | "대중교통으로 오시길 강추"만 반복, 주차 정보 완전 누락 |
| 음식 가격이 높을 가능성 — 가성비 기대 시 실망 위험 | 78% | 메뉴 가격·메뉴판 언급 없이 극찬만 나열 |

---

**비교: 진성 후기 분석 결과**

```
동네 단골 카페인데 오늘도 아메리카노 한 잔 했어요.
자리가 좁고 소음이 있어서 공부하기엔 별로인데, 커피 맛은 진짜 좋아요.
가격은 4500원으로 주변 대비 약간 비싼 편. 그래도 맛으로 계속 오게 되는 곳.
```

| 항목 | 결과 |
|------|------|
| 🟢 광고 확률 | **5%** — 진성 후기 |
| ⭐ 신뢰도 | 95점 |
| 📝 탈광고 요약 | 단점(좁은 자리·소음·가격)을 직접 언급한 신뢰도 높은 진성 후기입니다. |

> 광고 글과 진성 후기의 결과가 이렇게 극명하게 차이납니다.
