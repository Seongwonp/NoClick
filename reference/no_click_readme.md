# No-Click: 리뷰의 진짜 얼굴을 30초 안에

> **"텍스트가 있는 곳이면 어디든, 리뷰의 진짜 얼굴을 보여줍니다."**
> No-Click은 AI 기술을 활용하여 온라인 리뷰의 기만적인 광고 패턴을 탐지하고, 숨겨진 단점을 복원하며, 사용자 중심의 객관적인 정보를 제공하는 지능형 리뷰 분석 서비스입니다.

---

## 1. 프로젝트 배경 및 문제 정의
- **광고의 범람:** 협찬, 바이럴, 체험단 글이 폭증하며 사용자의 의사결정을 방해하고 있습니다.
- **정보의 비대칭:** "협찬 받음" 문구만으로는 글이 의도적으로 숨긴 실제 단점을 파악하기 어렵습니다.
- **사회적 비용:** 가짜 리뷰에 속아 낭비되는 시간과 비용을 줄이기 위한 범용적인 탐지 도구가 필요합니다.

## 2. 서비스 핵심 가치 (Core Values)
1. **Platform Agnostic (범용성):** 네이버 블로그, 쿠팡, 인스타그램 등 플랫폼에 상관없이 텍스트만 있다면 분석 가능합니다.
2. **Inference-Based Analysis (추론형 분석):** 단순 분류를 넘어 "AI 재작성"과 "숨긴 단점 복원"이라는 실질적인 해결책을 제시합니다.
3. **Visual Impact (시각적 직관성):** 3색 하이라이트 시스템과 비포/애프터 대비를 통해 분석 결과를 즉각적으로 체감하게 합니다.

## 3. 화면별 설계 명세 (Design Specification)

### 3.1 메인 랜딩 (The Gateway)
- **목적:** 사용자가 즉시 리뷰 분석을 시작할 수 있는 환경 제공.
- **주요 기능:**
  - 대형 텍스트 입력창 (모든 플랫폼 지원).
  - 플랫폼 감지 엔진 (URL 입력 시 자동 인식).
  - 핵심 차별점(패턴 인식, 신뢰 지수, 요약 리포트) 소개 카드.
- **디자인 전략:** 산뜻한 화이트 톤과 그린 포인트를 사용하여 '정화(Purification)'의 느낌을 전달.

### 3.2 하이라이트 분석 (The Truth Revealer)
- **목적:** AI가 리뷰를 분석하는 기술적 과정을 시각화하여 신뢰도 확보.
- **주요 기능:**
  - **3색 형광펜 시스템:** 
    - 🔴 광고 의심 문구
    - 🟡 과장/미화 표현
    - 🔵 핵심 키워드 및 패턴
  - 실시간 분석 로그: AI가 수행 중인 작업을 텍스트로 노출.
- **디자인 전략:** 실시간 진행률(Progress Bar)과 애니메이션 효과로 '철저한 분석' 중임을 시각화.

### 3.3 결과 리포트 (The Contrast)
- **목적:** 분석된 진실을 기반으로 사용자의 현명한 소비 결정 지원.
- **주요 기능:**
  - **Truth Score:** 리뷰의 진실도를 0~100% 수치로 시각화.
  - **AI 재작성:** 광고 수식어를 제거하고 객관적 사실만으로 정제된 리포트 제공.
  - **복원된 단점 카드:** 원문에서 교묘하게 가려졌던 단점(웨이팅, 가격, 공간 등)을 명확하게 도출.
- **디자인 전략:** '원본'과 '진실'의 대비를 명확히 하여 서비스의 효용성을 극대화.

## 4. 기술 스택 및 분석 로직
- **언어 모델:** LLM 기반의 텍스트 패턴 분석 및 맥락 추론.
- **분석 알고리즘:**
  - 감탄사/형용사 밀도 체크 (네이버 블로그 패턴).
  - 해시태그 및 특정 키워드 배치 분석 (인스타그램 패턴).
  - 앞뒤 문맥 간의 논리적 모순 탐지 (단점 복원 로직).
